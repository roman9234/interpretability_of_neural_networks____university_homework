По поводу форманты:

Необходимо будет посчитать это значение для каждой фонемы.
Желательный план работы - с помощью Montreal Forced aligner получить таймстемпы(временные метки) для каждой фонемы.
Этим первым этапом можно закрыть топик длина каждой фонемы.

С помощью этого иструмента вы получаете непрерывные значения, но тоже во временной шкале.
Далее вам надо будет разбить их по таймстемпам фонем и усредний для каждой фонемы.



Пример, у вас есть массив питча длиной 2 секунды. Вы знает, что первая фонемы была 0.00-1.09, а вторая 1.09-2.00.
Вы разбиваете этот массив питча на два массива, первый будет 0.00-1.09, а второй 1.09-2.00. Для каждого массива находите среднее значение.
Получилось, что для каждой фонемы теперь у вас есть значения
